How It Works

The calculator applies simplified duty percentages to CIF value and computes customs duty, PAL, excise duty, and VAT in sequence.

Current Sri Lanka Tax Rules

Real vehicle import taxes depend on periodic policy circulars, engine category, emissions profile, and concession status. This tool uses a transparent mock logic for planning and comparison only.

FAQ

Is this an official customs calculator?

No. It is a simplified estimate model and not a substitute for customs clearance documents.

Why do fuel type and engine capacity matter?

Duty structures typically vary by powertrain and engine class, so the model adjusts rates using those inputs.

Does this include registration fees?

No. Registration, compliance, logistics, and insurance costs are not included in this output.
